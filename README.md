# PowerPointPlus
A Grasshopper 3d plugin for PowerPoint File Read &amp; Write

_Like PowerPoint+ [Buy me a coffee :D](http://www.buymeacoffee.com/davidmans)_

---

This plugin for Grasshopper 3d makes it simple to construct Microsoft PowerPoint presentations without the need to have PowerPoint installed. Layout Text, Images, Tables, Charts, and Shapes into PowerPoint Slides and save the Presentation to a File or write to a Memory Stream. It is built on top ofÂ Shape Crawler.
